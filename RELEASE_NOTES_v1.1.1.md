## SoundShare v1.1.1

Hotfix for v1.1.0 startup and owner panel access.

**Developer:** H M Shahadul Islam

### Download

**`SoundShare-Setup-1.1.1.exe`** — single-file Windows installer (~67 MB)

### What's fixed

- **Host dashboard** (`/host`) opens on launch — shows Network URL, QR code, and listener count
- **Owner panel** works on the host PC without entering a token
- Remote panel access still requires the owner token from the console

### What's in v1.1

- Owner panel — track, remove, and block connected devices
- Optional listener password
- Device names for listeners
- Full VB-Cable bundled in one installer

### Quick start

1. Download and run **`SoundShare-Setup-1.1.1.exe`** as Administrator
2. Launch **SoundShare** — host dashboard opens automatically
3. Copy the **Network URL** and share with phones on Wi-Fi
4. Open **Panel** to manage devices or set a password
