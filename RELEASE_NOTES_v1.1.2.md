## SoundShare v1.1.2

Critical fix for installed builds (Program Files permission error).

### Download

**`SoundShare-Setup-1.1.2.exe`** — single-file Windows installer

### What's fixed

- **Server crash on install** — config now saves to `%LOCALAPPDATA%\SoundShare\` instead of `Program Files` (fixes Permission denied crash)
- Includes v1.1.1 host dashboard and owner panel fixes

### Quick start

1. Download and run **`SoundShare-Setup-1.1.2.exe`** as Administrator
2. Launch **SoundShare** from Desktop
3. Host dashboard opens with your Network URL
4. Share URL with phones on Wi-Fi
